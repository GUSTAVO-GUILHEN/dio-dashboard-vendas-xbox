# dio-dashboard-vendas-xbox# Dashboard de Vendas - Xbox Game Pass

## Objetivo
Este projeto apresenta um dashboard de vendas desenvolvido em Excel com foco na visualização e análise de assinaturas do Xbox Game Pass.

## Estrutura do Projeto

- `dashboard_xbox_gustavo.xlsx` → arquivo principal contendo:
  - Base de dados
  - Cálculos auxiliares
  - Dashboard interativo

## Indicadores disponíveis

- Receita total
- Quantidade de assinaturas
- Distribuição por plano
- Receita por tipo de assinatura
- Uso de Season Pass
- Análise visual consolidada

## Tecnologias Utilizadas

- Microsoft Excel
- Tabelas Dinâmicas
- Fórmulas
- Gráficos
- Segmentação de Dados

## Como reproduzir

1. Abra o arquivo `.xlsx`
2. Navegue até a aba `Dashboard`
3. Utilize os filtros e segmentações para explorar os dados

## Estrutura das Abas

| Aba | Função |
|---|---|
| Bases | Dados brutos |
| Cálculos | Métricas e agregações |
| Dashboard | Visualização final |
